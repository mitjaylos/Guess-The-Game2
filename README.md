## Guess the Game Screenshot

A small **offline-friendly** browser game:

- You see a **blurred game screenshot**
- Every **2 seconds** the blur reduces (from **20px → 0px**)
- Pick the correct game from **4 options**
- **10 rounds** per run
- **Faster guesses = more points**

### Run locally

- Open `index.html` in a browser (double click).

### Images

The `images/` folder contains lightweight **SVG placeholders** so the game works without downloads.
You can replace them with real screenshots later (just update the `image` paths in `script.js`).
